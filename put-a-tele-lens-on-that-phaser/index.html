Put a Tele Lens on that Phaser

Mar 14, 2015 #game #development #es6 #javaScript #phaser #guide #tutorial
I’ve been working with Phaser for a while now and I feel comfortable with the framework. However, recently I was making a game where need the camera to zoom in and out. That’s when I realized Phaser Camera does not have zoom functionality!!!

This article was written based on Phaser 2.3, some of the information regarding the framework might not be applicable on later version of Phaser.

I’ve been working with Phaser for a while now and I feel comfortable with the framework. Its API is pretty decent and simple, very easy to learn and performs well on mobile devices. It was smooth and happy for me.

However, recently I was making an action/JRPG game. I wish to utilize camera movement/zooming in order to create better effect and experience. That’s when I realized Phaser Camera does not have zoom functionality!!!

So I googled for some work around but none of the implementations looks cool to me. Their codes look just…messy and complicated. I just thought, why does it have to be complicated? So I decided to make my hands do some dirty work.

My way is simple: make a group and throw what ever I want to see on the camera in it and scale it when I wanna zoom the camera.

So this is what I came up with:

 1     class Camera extends Phaser.Group {
 2         constructor() {
 3             ...
 4             this.scale.setTo(1);
 5             ...
 6         }
 7         zoomTo(scale) {
 8             this.scale.setTo(scale)
 9         }
10     }

Aha, ain’t that easy?

Yes it is.

Does it work?

Not so fast.

When zooms out and pans, the camera pans outside of the group.

That is because as default, the camera bounds is set to the size of the world, when the group scales, the bounds should also be scaled and repositioned.

 1     constructor() {
 2         ...
 3         this.bounds = Phaser.Rectangle.clone(game.world.bounds);
 4         ...
 5     }
 6     zoomTo(scale) {
 7         var bounds       = this.bounds;
 8         var cameraBounds = game.camera.bounds;
 9         cameraBounds.x      = bounds.width  * (1 - scale) / 2;
10         cameraBounds.y      = bounds.height * (1 - scale) / 2;
11         cameraBounds.width  = bounds.width  * scale;
12         cameraBounds.height = bounds.height * scale;
13         ...
14     }

Finally, I added the ability to zoom slowly, because people like it smooth. 

 1     zoomTo(scale, duration) {
 2         ...
 3         if (!duration) {
 4             ...
 5         } else {
 6             game.add.tween(cameraBounds).to({
 7                 x      : bounds.width  * (1 - scale) / 2,
 8                 y      : bounds.height * (1 - scale) / 2,
 9                 width  : bounds.width  * scale,
10                 height : bounds.height * scale
11             }, duration).start();
12             return game.add.tween(this.scale).to({
13                 x: scale, y: scale
14             }, duration).start();
15         }
16     }
